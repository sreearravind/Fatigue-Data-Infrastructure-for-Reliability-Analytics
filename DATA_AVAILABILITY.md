The datasets, database schema-related scripts, ETL workflows, statistical analysis outputs, reliability analysis files, machine-learning validation outputs, selected figures, and manuscript-supporting result tables associated with this study are provided in this anonymous review repository. The repository contains an expanded and validated 85-sample-level fatigue dataset across 17 processing routes, together with processed cycle summaries, sample-level PSPP descriptors, bootstrap confidence intervals, Weibull reliability outputs, microstructure-property-performance correlation files, and route-aware machine-learning validation outputs. Supplementary Tables S1-S12 are available through the integrated manuscript-output tables and supporting CSV/Excel files. All author-identifying information has been removed to preserve double-blind peer review. Upon acceptance, the repository can be migrated to a permanent public archive such as GitHub/Zenodo with an assigned DOI.
